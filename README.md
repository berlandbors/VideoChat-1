# VideoChat+

**P2P видеочат с текстовым чатом, PWA и уведомлениями** — работает без собственного сервера, используя только [ScaleDrone](https://www.scaledrone.com/) для сигналинга.

---

## ✨ Возможности

### Видео и аудио
- **P2P видеосвязь** через WebRTC + Google STUN
- Управление микрофоном и камерой (вкл/выкл)
- **Переключение камеры** (фронтальная ↔ основная) на мобильных
- Перетаскиваемое PiP-окно локального видео

### Текстовый чат
- Встроенная панель чата поверх видео
- Разделение сообщений: «мои» (синие) и «собеседника» (серые)
- Метки времени у каждого сообщения
- Индикатор «Собеседник печатает…»
- **Счётчик непрочитанных** сообщений на кнопке чата
- `Enter` — отправить, `Shift+Enter` — новая строка
- **Автосохранение** истории в `localStorage` (последние 100 сообщений на комнату)
- История загружается при повторном входе в ту же комнату

### PWA
- **Установка** приложения на устройство (Android, iOS, Desktop)
- **Оффлайн-страница** при отсутствии сети
- Кэширование статических ресурсов через Service Worker

### Уведомления
- **Локальные браузерные уведомления** (Notification API) — работают когда вкладка в фоне
- Уведомления о событиях: новое сообщение, собеседник присоединился/покинул комнату
- **Звуковые сигналы** через Web Audio API: входящий звонок, сообщение, присоединение

## 🚀 Быстрый старт

1. Зарегистрируйтесь на [ScaleDrone](https://www.scaledrone.com/) и создайте канал
2. Скопируйте **Channel ID** из дашборда
3. Откройте приложение и введите Channel ID + название комнаты
4. Поделитесь ссылкой с собеседником (кнопка 🔗) — комната подставится автоматически
5. Нажмите 📞 для начала звонка

## 📱 Установка как PWA

### Android (Chrome)
- Откройте сайт в Chrome
- Нажмите «Добавить на главный экран» в меню или на всплывающем баннере

### iOS (Safari)
- Откройте сайт в Safari
- Нажмите кнопку «Поделиться» → «Добавить на экран Домой»

### Desktop (Chrome / Edge)
- В адресной строке нажмите иконку установки (📥)

## 🔔 Уведомления

Браузерные уведомления запрашиваются автоматически при первом запуске. Для повторного запроса нажмите кнопку 🔔 на панели управления.

> **Важно:** уведомления работают только когда браузер запущен. Для уведомлений при закрытом браузере потребовался бы собственный Push-сервер.

## ❓ FAQ

**Почему нужен ScaleDrone?**  
WebRTC требует сервер для первоначального обмена SDP и ICE-кандидатами. ScaleDrone предоставляет бесплатный pub/sub канал для этого.

**Сколько участников поддерживается?**  
Сейчас два участника (P2P). Для группового чата потребовался бы SFU/MCU сервер.

**Сохраняются ли сообщения на сервере?**  
Нет. История хранится только в `localStorage` вашего браузера.

**Почему уведомления не приходят когда браузер закрыт?**  
Для этого нужен бэкенд-сервер (Web Push API). Текущая реализация — без сервера.

**Работает ли приложение без интернета?**  
После первой загрузки форма входа будет доступна оффлайн. Видеосвязь и чат требуют активного соединения с ScaleDrone.
